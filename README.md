# Online Train Ticket Booking App

This project is an Android-based application developed using Kotlin for online train ticket booking services. The application was created as part of Android mobile development learning activities and implements various Android concepts ranging from UI components, Activity navigation, Material Design, local storage, API integration, until machine learning implementation using ONNX.

The project demonstrates the implementation of modern Android application development using Material Components, ViewBinding, SharedPreferences, Retrofit, Room Database, and several advanced Android features.
